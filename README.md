## 特点

-   日志全局开关
-   日志全局标签TAG
-   日志全局开关
-   日志全局拦截器
-   打印日常信息
-   输出JSON
-   输出超长字符串
-   日志堆栈输出
-   自定义日志等级
-   Kotlin特性

## 安装

project 的 build.gradle

```groovy
allprojects {
    repositories {
        // ...
        maven { url 'https://jitpack.io' }
    }
}
```



module 的 build.gradle

```groovy
implementation 'com.github.liangjingkanji:LogCat:1.1'
```
